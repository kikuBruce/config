### vim plugin

###### 目录树
###### ctrlp
1. 使用ctrl + p调用全局搜索
2. ctrl + f 或者 ctrl + b 切换 buffer/mru file最近访问/file
3. ctrl + d 切换目录搜索和文件名搜索
4. ctrl + z 标记文件， ctrl + o 全部打开
5. ctrl + r 切换re匹配

###### 代码补全
###### jedi
1. 使用tap补全代码


###### vim buffer
1. 使用:ls 查看buffer list
2. 使用:b [1] 切换到对应的buffer
3. 使用:bd [1] 删除1号buffer -- buffer delete 1

##### 函数跳转
1. ctrl + ] 跳转光标所在方法定义
2. ctrl + t 跳转回所在文件
